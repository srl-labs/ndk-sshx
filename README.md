# ndk-sshx

[sshx.io](https://sshx.io) tunneling in SR Linux with [NDK](https://learn.srlinux.dev/ndk/).


https://github.com/user-attachments/assets/3a71043e-8b66-45a4-a291-f93a3366df12


Download sshx bin:

```bash
./run download-sshx
```

then run:

```bash
./run deploy-all
```

to start the lab.
